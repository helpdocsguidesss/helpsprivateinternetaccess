Troubleshooting Tips Before Contacting PIA Technical Support
-------------------------------------------------------------

Sometimes simple steps can fix common issues without needing to contact Private Internet Access help center.

Switch VPN Servers  
~~~~~~~~~~~~~~~~~~~

Changing to a different server location often resolves slow speed or connection issues.

Change VPN Protocol  
~~~~~~~~~~~~~~~~~~~~

Private Internet Access VPN allows users to switch between different protocols. Trying another protocol may improve stability.

Disable Conflicting Software  
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Firewall or antivirus software can sometimes interfere with VPN connections. Temporarily adjusting settings may solve the issue.

Reinstall the Application  
~~~~~~~~~~~~~~~~~~~~~~~~~~

Uninstalling and reinstalling the PIA VPN app can fix corrupted installation files.

If these steps do not solve the problem, contacting Private Internet Access customer support is the best next step.
